Overview

NSPoint additions.

Examples:

point = OSX::NSPoint.new(42, 24)
=> #<OSX::NSPoint x=42.0, y=24.0>
point.in? OSX::NSRect.new(40, 20, 2, 1) # x, y, w, h
=> false
point.in? OSX::NSRect.new(40, 20, 3, 2)
=> true
